# Configurações

Este diretório contém os arquivos de configuração para o projeto `noshow-prediction-ml`. Ele permite separar os parâmetros de configuração do código, facilitando a gestão de diferentes ambientes (desenvolvimento e produção).

## Arquivos:

- `local.yaml`: Contém as configurações específicas para o ambiente de desenvolvimento local.
- `prod.yaml`: Contém as configurações para o ambiente de produção, otimizadas para implantação.
